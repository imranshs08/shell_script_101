## üìù Cheatsheet: Common Patterns

```bash
# Shebang
#!/bin/bash

# Debug Mode
set -x  # Print commands
set -e  # Exit on error

# Variables
NAME="DevOps"
echo "Hello $NAME"

# Arguments
SCRIPT_NAME=$0
FIRST_ARG=$1

# Condition
if [ -f "/etc/passwd" ]; then
    echo "File exists"
fi

# Loops
for file in *.log; do
    echo "Processing $file"
done

# Function
check_status() {
    systemctl is-active $1
}
```

## üíº Interview Prep: Top Questions

1.  **Beginner**: "How do you make a script executable?" (`chmod +x script.sh`)
2.  **Intermediate**: "What is the difference between `$?` and `$!`?" (`$?` = last exit code, `$!` = last background PID).
3.  **Real-World**: "How do you ensure only one instance of your script runs at a time?" (Lock files).
4.  **Scenario**: "I have a log file with 1 million lines. How do I get line 500 to 510?" (`sed -n '500,510p' file` or `head` + `tail`).
